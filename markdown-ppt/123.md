# 實驗一<br>簡單LED電路
利用單獨arduino控制一顆LED閃爍十次
![bg right:40%](1.png)

---
# 程式碼
```arduino
int LED =12;//輸出的pin位
void setup() {
  Serial.begin(9600);
  pinMode(LED,OUTPUT);//設定輸出
}
void loop() {
    for(int i=0;i<10;i++){
        digitalWrite(LED,1);
        delay(100);
        digitalWrite(LED,0);
        delay(100);
        Serial.println(i,DEC);//輸出現在是第幾次閃
    }
    Serial.println("ALL DONE!!");
    Serial.println("全部完成!!");
```
